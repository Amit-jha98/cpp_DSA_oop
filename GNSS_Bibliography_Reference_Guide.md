## How to Use These References

### For Your Literature Review
1. **Start with foundational texts** [1-2] to understand GNSS basics
2. **Read error modeling papers** [3-21] to understand traditional approaches
3. **Study ML applications** [22-33] to see state-of-the-art
4. **Review ML fundamentals** [34-37] if needed

### For Citing in Your Paper
Always cite:
- When stating facts about GNSS errors
- When using traditional models for comparison
- When building on others' ML approaches
- When using standard algorithms (LSTM, RF, etc.)

### Accessing Papers

**Open Access (Free):**
- References [25, 26, 27, 30]: MDPI journals
- Reference [37]: arXiv preprint
- Reference [34]: Free online version

**University Access:**
- Most papers accessible through university library
- Use IEEE Xplore, SpringerLink, Wiley subscriptions

**If No Access:**
- Try Google Scholar → PDF links
- Email authors directly (usually respond)
- Use ResearchGate
- Request via interlibrary loan

## Common Citation Mistakes to Avoid

1. ❌ Citing papers you haven't read
2. ❌ Using only recent papers (ignore classics)
3. ❌ Using only old papers (ignore recent work)
4. ❌ Incorrect author names or initials
5. ❌ Wrong page numbers or DOIs
6. ❌ Inconsistent formatting
7. ❌ Missing DOIs when available
8. ❌ Citing Wikipedia or non-peer-reviewed sources
